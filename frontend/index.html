What is a Convex Hull?

A convex hull is the smallest convex shape that contains all given points. Think of it as stretching a rubber band around all the points.

1. Generate Points: Create random points or click on the canvas
2. Choose Algorithm: Select which convex hull algorithm to explore
3. Run & Compare: Execute algorithms and see step-by-step visualization
